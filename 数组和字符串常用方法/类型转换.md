#隐式数据类型转换
1. 运算时
```javascript
  //只有数值类型才能进行运算，最终运算结果肯定是数值类型  将非数字 用 Number 方法 转成数字
  console.log(1-'1')         // 0
  console.log(1-'1a')       // NaN
  console.log(1-undefined)   // NaN
  console.log(1-'aa')       // NaN
  console.log(1-null)        // 1
  console.log(1-true)       // 0
  console.log(1-[])         // 1
  console.log(1-[2])        //  -1
  console.log(1-[2,3])      // NaN
  console.log(1-{})         // NaN
  console.log(1-{name:'lsy'})  // NaN
```
